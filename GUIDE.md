# 快速使用指南

> 用这个工具生成有设计感的演示文稿，浏览器打开即用，零安装。

## 30 秒上手

1. 告诉我你的主题、观众、页数
2. 我会生成一个 HTML 文件
3. 双击打开 → 方向键翻页，`F` 全屏，`S` 演讲者备注

就这么简单。

## 怎么描述需求

好的描述 = 我一次就能生成你满意的东西。

### 最简方式

```
做个 PPT，主题是 XXX，给 XXX 看的，大概 X 页
```

### 更好的方式

```
做一个 8 页的产品发布演示文稿
主题：我们新上线的 AI 写作助手
观众：技术团队（30 人），了解基础 AI 概念
语言：中文
需要：1 页产品架构图、1 页竞品对比、结尾要有行动号召
```

### 可以指定的选项

| 选项 | 说明 | 不指定则自动选择 |
|------|------|----------------|
| 主题 | 演示的核心内容 | 必须提供 |
| 观众 | 谁会看这个 PPT | 默认通用 |
| 页数 | 多少页 | 默认 6-10 页 |
| 语言 | 中文/英文 | 默认中文 |
| 模板 | 5 种视觉风格（见下） | 根据主题自动匹配 |
| 色彩 | 指定主色调 | 根据场景自动选择 |

## 5 种模板风格

| 风格 | 一句话描述 | 适合场景 |
|------|-----------|---------|
| **Editorial** | 优雅、正式、像杂志 | 学术报告、商务汇报 |
| **Dark Tech** | 高对比、终端感 | 产品发布、开发者大会 |
| **Minimal** | 网格、空间感 | 架构说明、解释性内容 |
| **Vibrant** | 大胆、活力 | 营销推广、创意展示 |
| **Nature** | 柔和、清新 | 教学、内部分享 |

不用记名字——告诉我场景，我会自动匹配。

## 常用操作

### 导出

| 格式 | 方法 |
|------|------|
| **PPTX**（可编辑） | 打开 HTML → 左下角按钮 → 点击即下载 |
| **PDF** | Chrome 打开 `file.html?print-pdf` → `Ctrl/Cmd+P` → 打印为 PDF |

### 导航快捷键

| 快捷键 | 功能 |
|--------|------|
| ← → | 上下翻页 |
| ↓ ↑ | 垂直子页（如果有） |
| `F` | 全屏 |
| `S` | 演讲者备注窗口 |
| `ESC` | 幻灯片总览 |

### 修改内容

HTML 是纯文本，可以用任何编辑器打开修改：
- **改文字**：直接搜索替换文案
- **改颜色**：找到 `:root` 中的 `--c-accent`、`--c-bg` 等变量
- **改图片**：替换 `<img src="...">` 的链接
- **加减页面**：复制或删除 `<section>...</section>` 块

## 验证质量

生成后会自动运行设计规范检查。你也可以手动运行：

```bash
node scripts/lint-design.js 你的文件.html
```

- `P0 = 0` = 通过，可以直接使用
- 有 P0 违规 = 需要修复

## 常见问题

**Q: 需要安装什么吗？**
A: 不需要。HTML 文件通过 CDN 加载依赖，浏览器打开就行。

**Q: 能在手机上看吗？**
A: 可以，但体验不如电脑。建议导出 PDF 后发送到手机。

**Q: 怎么分享给不会用 HTML 的人？**
A: 导出 PPTX，对方用 PowerPoint/WPS 打开编辑。

**Q: 生成的 PPT 太长了，能缩减吗？**
A: 可以。告诉我缩减到多少页，我会优先保留核心内容。

**Q: 不满意设计风格？**
A: 告诉我哪里不满意（颜色太暗、字体不喜欢、布局太简单等），我会针对性调整。也可以直接指定换一个模板。

## 文件结构（供高级用户参考）

```
生成文件.html          ← 你的演示文稿（自包含 HTML）
scripts/
  lint-design.js       ← 设计规范检查
  validate.js          ← 溢出检测 + 截图
  export-pptx-client.js ← PPTX 导出脚本（生成 HTML 时默认内联）
examples/              ← 5 个模板示例
references/            ← 设计参考资料
```
