# 案例 07 · Brutalist(粗野编辑)

> 来源 · Brutalism 设计运动(建筑 → Web → Print)+ 当代品牌应用(Balenciaga / Bloomberg Businessweek / Gucci 等)
> 类型 · 粗野编辑 / 反精致
> 风格标签 · Brutalist
> 公认度 · 当代设计反潮流标杆,高端品牌(Balenciaga/Gucci)与媒体(Bloomberg Businessweek)采用,Awwwards 常见 Brutalist 分类
> 官方对照 · [Balenciaga 官网](https://www.balenciaga.com/)(品牌粗野:系统字 + 暴露栅格 + 冲突色)·[Awwwards Brutalist 合集](https://www.awwwards.com/websites/brutalism/)(看粗野 web 设计全集)

## 满分维度(6 维 rubric)

- Innovation ⭐⭐⭐⭐⭐ — 反精致即签名,反主流审美
- Visual ⭐⭐⭐⭐ — 高对比 + 原始元素
- Cohesion ⭐⭐⭐⭐ — 一套粗野规则贯穿
- Communication ⭐⭐⭐⭐ — 直白即沟通,无修饰
- TechnicalCraft ⭐⭐⭐ — 刻意"粗糙"也是工艺
- Audience ⭐⭐⭐⭐ — 反主流/高端品牌受众

## 为什么好(3 个决定)

1. **反精致即风格** — 系统字体(Helvetica/Arial)/ 暴露栅格 / 高对比黑白 / 无圆角无阴影。刻意的"粗糙"是签名,不是失误。
2. **内容优先到极致** — 没有任何装饰,只有内容和裸结构。排版即全部设计。
3. **高对比冲击** — 纯黑底白字 / 纯白底黑字 + 单一荧光色(酸绿/电粉)。视觉冲击来自对比度,不是渐变或光效。

## 可借鉴技法(生成时调用)

- 暴露栅格:网格线可见,不对齐到"精致",刻意的原始
- 系统字体(Helvetica/Arial/系统 sans)+ 极端字阶(超大 hero vs 超小标注)
- 高对比:纯黑白 + 一个荧光色,不渐变(对齐 [technical-specs.md](../technical-specs.md) 配色章节)
- 无圆角、无阴影、硬边;按钮就是文字链 + 边框
- 信息密度可高,不加"卡片"美化

## 反 AI 味点

- ✗ 没有渐变 / 圆角 / 阴影
- ✗ 没有"精致"装饰
- ✗ 没有 glassmorphism / 发光
- ✓ 刻意粗糙 = 反 AI 审美(AI 默认追求"精致",Brutalist 反其道)

## 对应我们的模板

**template-06 brutalist** — 共享:粗野 + 高对比 + 系统字体
差异:我们的 brutalist 已是这个方向,可往 Balenciaga 的"品牌粗野"收——加冲突荧光色 + 暴露栅格。生成反潮流/潮流品牌 deck 时用这条线。

## 一句话

> 粗糙即精致,对比即冲击,内容即设计。
