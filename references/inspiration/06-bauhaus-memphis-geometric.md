# 案例 06 · Bauhaus / Memphis(几何原生形式)

> 来源 · Bauhaus 设计学校(1919–1933,公开历史设计运动)+ Memphis Group(1980s 米兰,公开后现代设计)
> 类型 · 设计运动 / 几何原生
> 风格标签 · Bauhaus/Geometric × Memphis
> 公认度 · 现代设计教育源头(Bauhaus)+ 后现代设计标杆(Memphis),设计史必修课
> 官方对照 · [Bauhaus 100](https://www.bauhaus100.de/)(百年纪念,几何/原色系统)·[Memphis Milano](https://www.memphis-milano.com/)(官方站点,看不对称几何 + 波普色)

## 满分维度(6 维 rubric)

- Innovation ⭐⭐⭐⭐⭐ — 几何即形式,原创性的历史源头
- Visual ⭐⭐⭐⭐⭐ — 原色 + 几何 + 网格的极致简化
- Cohesion ⭐⭐⭐⭐⭐ — 一套几何规则贯穿所有元素
- TechnicalCraft ⭐⭐⭐⭐ — 工艺即设计(Bauhaus 核心信条)

- Audience ⭐⭐⭐ — 设计/艺术受众为主

## 为什么好(3 个具体决定)

1. **几何即语言** — 圆/方/三角 + 红/黄/蓝原色,简化到极致。Kandinsky/Itten 的"色彩—形状对应"理论把视觉元素降维到最小集。不是装饰,是底层语法。
2. **形式追随功能 → 形式即功能** — Bauhaus 把"用途"和"外观"统一,消灭装饰。Memphis 反叛——形式本身就是内容,不对称 + 波普色 + 戏谑图案,反一切"得体"。
3. **网格系统源头** — 所有元素严格对几何网格。是瑞士设计/现代排版/Tufte 数据可视化的共同源头。

## 可借鉴技法(生成时调用)

- 几何 deck:用圆/方/三角作为版面元素,不用图标/插画
- 原色系统:红/黄/蓝 + 黑白,不做渐变(对齐 [technical-specs.md](../technical-specs.md) 配色章节)
- Memphis 风:不对称几何拼贴 + 波普色 + 装饰性图案(点阵/条纹/之字),反"科技感模板"
- 网格严格对齐 8 的倍数,网格线可暴露
- 字体用几何无衬线(Futura / Helvetica 渊源),不用人文衬线

## 反 AI 味点

- ✗ 没有渐变 mesh / glassmorphism
- ✗ 没有"科技感"indigo→紫配色
- ✗ 没有 emoji / 通用 outline 图标
- ✓ 几何原色 = 反 AI 模板的终极武器(模板做不出这种克制)
