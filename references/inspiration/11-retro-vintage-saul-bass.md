# 案例 11 · Saul Bass × Synthwave(复古时代美学)

> 来源 · Saul Bass 经典平面设计(希区柯克/库布里克电影海报)+ Paul Rand(IBM/ABC logo)+ 80s Synthwave(霓虹粉紫 + 网格地平线)+ Risograph 印刷美学
> 类型 · 复古 / 时代美学
> 风格标签 · Retro/Vintage
> 公认度 · Saul Bass / Paul Rand 是平面设计史经典,复古风是品牌差异化的常用策略。Synthwave 是当代数字设计的复古回潮。**反 AI 的有力武器——AI 默认出"现代科技感",复古反其道**
> 官方对照 · [Saul Bass 海传集](https://www.saulbass.net/)(看极简几何 + 一图形主隐喻的电影海报式构图)·搜 "synthwave design"(看 80s 霓虹粉紫 + 网格地平线 + 日落渐变)

## 满分维度(6 维 rubric)

- Innovation ⭐⭐⭐⭐⭐ — 时代美学即差异化,反主流
- Visual ⭐⭐⭐⭐⭐ — 印刷质感 + 时代配色冲击
- Cohesion ⭐⭐⭐⭐⭐ — 一套时代语言贯穿
- Audience ⭐⭐⭐⭐ — 怀旧/小众品牌受众
- Communication ⭐⭐⭐ — 风格优先于叙事
- TechnicalCraft ⭐⭐⭐⭐ — 印刷质感本身是工艺

## 为什么好(3 个决定)

1. **时代配色锁定** — 每个年代有标志性配色:50s 焦橙 + 蓝绿、80s 霓虹粉紫、90s 大胆撞色。配色即时代标签,锁定一个年代不混。
2. **印刷质感** — Risograph / 活版 / 半色调网点,模拟物理印刷的"不完美"。质感是反"数字感"的核心——AI 默认出干净渐变,印刷质感反其道。
3. **时代字体** — 50s 流体 script + 粗衬线、80s 像素 + 几何无衬线、90s Grunge。字体即年代,不能用当代无衬线替代。

## 可借鉴技法(生成时调用)

- **锁定一个年代 + 配色**(50s 焦橙蓝绿 / 80s 霓虹粉紫 / 90s 撞色),不混年代
- **印刷质感**:半色调网点 / 套色偏移 / 纸纹理,反"数字渐变"
- **时代字体**(流体 script / 粗衬线 / 像素),不混当代无衬线(对齐 [technical-specs.md](../technical-specs.md))
- **Saul Bass 模式**:极简几何 + 一图形主隐喻,电影海报式 cover
- **复古图标手绘**(对齐 [icon-system.md](../icon-system.md)),不用通用 outline 图标
- **Synthwave 模式**:霓虹粉紫 + 网格地平线 + 日落渐变 + 80s 字体(只在主题匹配时用,否则 AI 味)

## 反 AI 味点

- ✗ 没有"现代科技感"indigo 渐变
- ✗ 没有"当代精致"圆角 + 玻璃拟物
- ✓ 印刷不完美 = 反数字反 AI 的核心
- ⚠ Synthwave 本身易滑向 AI 味,必须配时代字体 + 印刷质感才成立

## 我们的实现(这个风格的子集)

**当前无复古 template**。这是「风格 > template」的典型——Retro 风格存在且是反 AI 的有力武器(AI 默认出"现代科技感"),我们 template 没实现。做复古 deck 时**必须先锁定年代 + 配色 + 字体 + 印刷质感**,四件套缺一即 AI 味。后续可探索 template-11 retro。

## 一句话

> 锁定一个年代,印刷质感反数字,时代字体即标签。
