## 满分维度(6 维 rubric)

- Communication ⭐⭐⭐⭐⭐ — 一页一 idea,action title 极致
- Audience ⭐⭐⭐⭐⭐ — 现场观众理解力测试通过
- Visual ⭐⭐⭐⭐ — 大字 + 留白
- Cohesion ⭐⭐⭐⭐ — TED 红单一强调贯穿
- Innovation ⭐⭐⭐ — 克制即风格
- TechnicalCraft ⭐⭐⭐ — 简单但精确

## 为什么好(3 个决定)

1. **一页一 idea** — 每页只有一个论点,不堆砌。论点用完整短句(action title),不是 keyword 标签。
2. **大字 + 极致留白** — 标题占版面 50%+,剩下留白。没有装饰元素干扰焦点。
3. **TED 红单一强调** — 全 deck 用一个红(#E62B1E)做强调,其他全黑白。色彩克制是 TED 视觉签名。

## 可借鉴技法(生成时调用)

- 演讲 deck 强制一页一论点,action title 是完整短句(对齐 [content-budget.md](../content-budget.md))
- 标题用 `clamp(4rem, 7vw, 7rem)`,留白占 40%+
- 单一品牌色 + 黑白,不做多色系
- 论点页不放图,只用字;图只在"证据页"出现
- 转场用黑底白字大词,不用通用"章节分割线"

## 反 AI 味点

- ✗ 没有装饰性图标/插画
- ✗ 没有 gradient text
- ✗ 没有"信息图"堆砌
- ✓ 克制就是风格
