PLATFORM / 02 · OVERVIEWISO · ALL LAYERS

三层各司其职：Edge 接入、Mesh 编排、Data 持久。

EDGE LAYER

全球 28 个 PoP，TLS 终止 + 鉴权 + 限流。P99 延迟 12ms。

MESH LAYER

服务网格，gRPC + sidecar，自动熔断与重试。无单点。

DATA LAYER

分片 KV + 列存 + 流。10PB，每日全量校验。

每一层可以独立演进、独立扩缩容、独立出故障——而不影响其他两层的契约。
